# CodeAlpha_StudentGradeTracker
java console-based student grade tracker project
PROJECT DESCRIPTION

A Java console-based application developed during the CodeAlpha Java Internship.

FEATURES

- Student Marks Management
- Grade Calculation
- Pass/Fail Status
- Highest & Lowest Marks
- Average Marks Calculation
- Report Generation
- File Handling using report.txt

TECHNOLOGIES USED

- Java
- VS Code

OUTPUT

The application generates:

- Console report
- report.txt file automatically
